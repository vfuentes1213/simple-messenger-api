# Simple Messenger API
This is a simple messenger API used to send and retrieve messages from recipients/senders.
## Available scripts

### set up
```bash
make bootstrap-api
```
### start the server
```python
python run.py
```
### run unit tests/coverage report
```bash
make run-coverage-report
```
